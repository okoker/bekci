Backend (Go 1.25)

  • • net/http stdlib router (Go 1.22 method routing)
  • • go-sqlite3 1.14.37 (CGO_ENABLED=1 required)
  • • golang-jwt/v5 5.3.1 · bcrypt cost 12

  • • Vue ^3.5.25 · Vite ^7.3.1 (build only)
  • • vue-router ^4 · pinia ^3 · axios ^1.13

  • • Built bundle: 269 KB raw / 88 KB gzipped
  • • Frontend dist embedded in Go binary via //go:embed

Security & Auth

  • • JWT HS256 · HttpOnly cookie (token) · path /
  • • bcrypt cost 12 password hash
  • • Default first-boot creds: admin/admin1234 (user changes)
  • • Nginx headers: CSP · HSTS · X-Frame · X-CTO · Referrer
  • • Watchdog: process sends WATCHDOG=1 every 30s
  • • Fail2Ban integration · live + historical (SQLite)
  • • Backups: Argon2id KDF + AES (encrypted variant)

Data Architecture (A-011)

  • • SQLite 3 WAL · SetMaxOpenConns(1) · schema v24
  • • check_state: ~1.5K rows · current status (1:1)
  • • check_daily_rollups: ~135K rows (90d) · uptime
  • • check_results: ~6.5M rows (3d) · raw history
  • • Triple write in 1 transaction per result save
  • • Hourly purge: raw 3d · rollups 90d
  • • Auto backup: weekly/10day/monthly schedules

Checks & Alerts

  • • Probe types: HTTP/S · TCP · ICMP · SNMP · DNS · TLS
  • • ICMP requires CAP_NET_RAW (Docker cap_add / systemd ambient)
  • • SNMP settings cache: 15m TTL · all 7 keys together
  • • HTTP pool: 100 max idle · 5/host · 90s · TLS LRU 128
  • • Rule engine: per-rule mutex · atomic CAS · sem cap 10
  • • Alert channels: Resend email · Signal CLI · webhook
  • • System alerts: scheduler dead · crash detect · unclean restart

Operations & Deploy

  • • Local: Docker multi-stage build · Alpine 3.23 runtime
  • • Prod: bare-metal Ubuntu 22.04 · systemd · nginx :443
  • • Deploy: ssh + git pull + go build + systemctl restart
  • • Frontend pre-built · cmd/bekci/frontend_dist/ in git
  • • systemd WatchdogSec=120 · auto-restart on hang
