Zero false positives

The stars must align.

Most monitors panic at a single failed check. Bekci demands corroboration. Draw your checks as a constellation — the alert star only ignites when the map is truly satisfied.

Signals confirming
Rule  (TCP AND HTTP) OR PING → Stars aligned · alert fires
01 / Corroboration

Signals must agree

Combine checks with boolean logic. An alert fires only when independent signals corroborate — groups join by AND/OR internally, OR across groups, nestable and composable, like (TCP AND HTTP) OR PING.

02 / Sustained failure

Hold the line

A condition must fail N times within a window before it counts as down. A single blip is ignored — the window minimum is fail_count × interval, so one flickering star never wakes the house.

03 / Flap suppression

No alert storms

Per-rule alert cooldown, a configurable re-alert interval for issues still firing, and a recovery cooldown — a flapping target can't turn one incident into a hundred pages.

Eight points of light

Every kind of check.

Eight probe types, each yielding a clean pass or fail. Compose them into the constellations above.

ICMPPing

Reachability with round-trip time and packet loss. Passes as long as loss isn't total.

HTTP / HTTPSWeb endpoint

Expected status code on a path and custom port. GET, follows redirects.

TCPPort connect

Opens a TCP connection to confirm the port is listening and reachable.

DNSResolution

Resolves a host, with optional expected record value, record type and nameserver.

SHA-256Page Hash

Hashes the response body to catch defacement or any unexpected change.

TLSCertificate

Watches certificate expiry and warns a configurable number of days ahead.

SNMP v2cDevice health

Reads sysUpTime and sysDescr, with best-effort CPU and memory.

SNMP v3Secure SNMP

The same device health with authPriv, authNoPriv or noAuthNoPriv.

Deploy the watchman

One container.
One port. Go.

Clone, compose, open. The whole watch is a single self-contained Go binary with the Vue UI embedded, storing to SQLite.

Single Go binary — Vue 3 UI embedded
SQLite in WAL mode — no external database
One container, one port, scales to ~1000 hosts
MIT licensed — yours to self-host

Then open http://localhost:65000

watchman — deploy
# clone the watch
$ git clone https://github.com/okoker/bekci.git
$ cd bekci
$ docker compose up -d
 
✓ bekci up on :65000 — the watch begins
